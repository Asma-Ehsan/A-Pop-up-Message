# Popup Message Modal

A simple and beginner-friendly JavaScript project that displays a popup message when a button is clicked. This project focuses on understanding how UI elements can be shown and hidden using JavaScript and CSS transitions.

## Live Demo
Try it live here:  
https://github.com/Asma-Ehsan/A-Pop-up-Message

## Features
- Button-triggered popup modal
- Smooth open and close animation
- Centered popup layout
- Clean and responsive design
- Built using plain HTML, CSS, and JavaScript

## How It Works
- Clicking the **Submit** button adds a CSS class to the popup.
- The added class changes visibility, position, and scale.
- Clicking **OK** removes the class and hides the popup again.

## How to Use
1. Open `index.html` in your browser.
2. Click the **Submit** button.
3. A popup message appears with a confirmation message.
4. Click **OK** to close the popup.

## What I Learned
- How to show and hide elements using `classList.add()` and `classList.remove()`
- Using CSS transitions for smooth animations
- Positioning elements with `absolute` and `transform`
- Connecting JavaScript logic with UI behavior
- Importance of small projects to strengthen JavaScript fundamentals
